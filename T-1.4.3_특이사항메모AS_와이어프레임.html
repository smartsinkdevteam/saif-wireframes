명세 반영 사항 (기능 명세서 T-1.4 · T-1.4.3 기준)

  1. 1 특이사항 사진: 선택. 분류 "시공 특이사항"으로 해당 품목에 저장. 장당 10MB·분류별 30장·JPG/PNG/HEIC.
  2. 2 시공 메모: 필수, 1–1,000자. 시공 내용·특이사항을 함께 기록.
  3. 3 AS 필요: 체크박스, 선택(기본 미체크). AS 필요 시 어느 부위인지 시공 메모에 기재(명세 비고). "AS 완료"·AS 처리 프로세스는 1차 범위 아님(추후, 기사 재배정).
  4. 4 [저장] 활성 조건: 시공 메모 입력 + 2단계 시공 완료 사진 1장 이상 → 해당 품목 시공 정보 저장 → 품목 "시공완료" 처리 → [T-1.1.1 주문 상세] 복귀.
  5. 5 "시공 메모를 입력해주세요." — 명세 문구 그대로.
  6. 6 조건 미충족 시 [저장] 비활성.
  7. 7 [취소]: 입력 내용 있으면 확인 모달 후 주문 상세 복귀 — 임시 저장(draft)분 유지, 재진입 시 [이어서 (n/N)]로 마지막 단계부터 재개(1차 포함). 모달은 위저드 1~3단계 공통.
  8. 8 주문 단위 완료 판정: 모든 품목 시공완료 시 [T-1.1.1]의 [시공 완료] 활성 — 명세 그대로(안내 배너 자체는 임의).
  9. 9 에러 4종: 시공 메모 / 시공 완료 사진 / 사진 용량·장수 / 저장 실패 — 명세 문구 그대로.

⚠️ 문서에 없는 임의 표현

  1. 취소 모달 문구·버튼명("계속 입력"/"나가기"), AS 체크 하단 보조 설명, 시공완료 안내 배너, 메모 placeholder·글자 수 카운터는 임의 — 확정 필요.
  2. 메모 예시 텍스트·사진 구성은 가상의 예시.
